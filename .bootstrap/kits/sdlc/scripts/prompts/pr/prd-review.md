# Requirements Review Prompt

Review the following PR changes against requirements/PRD best practices.

Focus on:
- Completeness and clarity of requirements
- Testability and acceptance criteria
- Traceability to business goals
- Compliance with `{template}` template
- Ensure requirements are aligned with the best industry standard practices in `{project_domain}`
- Ensure this problem is not already solved by other existing PRDs in `{existing_artifacts}`

- Criticize requirements
- Split requirements by different topics and rate every 1-10

Use `{checklist}` as the structured review guide.
Pay attention to the "PR Review Focus (Requirements)" section at the end of the checklist.
